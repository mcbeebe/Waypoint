👤 User Actions & Touchpoints
  • Searches "help for child with autism"
  • Finds Waypoint via advocacy org referral, Google, or pediatrician
  • Downloads app or visits website
  • Reads landing page
  • Creates account (Google/Apple SSO)
  • Enters child's diagnosis, age, zip code
  • Sees Regional Center match
  • Selects preferred language
  • Reviews profile summary
  • Receives personalized action plan
  • Completes first 2-3 steps
  • Browses resource library
  • Bookmarks relevant resources
  • Sets up push notifications
  • Uploads IEP document for AI review
  • Uses email templates to contact school
  • Syncs Google Calendar
  • Manages therapy appointments
  • Tracks action plan progress
  • Joins community forums
  • Connects MyChart for medical sync
  • Tracks insurance authorizations
  • Uses legal resource guides
  • Manages multiple children's plans
  • Suggests resources for other parents
  • Answers community questions
  • Shares Waypoint with other families
  • Provides testimonials
  • Advocates for platform to Regional Centers
🔥 Pain Points & Barriers
"I just got the diagnosis and I'm terrified. I don't know what any of these acronyms mean."
"Do I trust this app with my child's private info? Is this legit?"
"Will this actually help, or is it just another generic resource list?"
"The IEP process is so adversarial. I'm afraid to push back on the school."
"I have so many systems to manage. It's a part-time job on top of everything else."
"I want to help others but I worry about sharing advice that might not apply to their situation."
🧭 Waypoint Features & Solutions
SEO Referral Landing

Clear value prop, social proof from parent testimonials, advocacy org partnerships, multilingual landing pages
Onboarding RC Lookup SSO

5-step wizard, zip-to-RC matching, optional income for resource filtering, privacy-first messaging
Action Plans Resources Push

AI-generated journey map, curated resource library with filters, first actionable step within 60 seconds
IEP Review Templates Calendar

AI IEP analysis with legal citations, pre-written email templates, Google Calendar sync, document management
Community MyChart Insurance

Peer forums, healthcare integration, authorization tracking, legal guides, multi-system dashboard
Contribute Refer Provider

Community contributions, resource suggestions, referral program, provider portal, verified expert badges

Emotional Journey

Journey Stage → Development Phase Mapping

Phase 1: MVP (Wk 1–10)
Covers journey stages 1–3: Awareness through First Value. Onboarding, action plans, resource library, RC lookup, basic notifications.
Phase 2: Intelligence (Wk 11–20)
Covers journey stage 4: Active Use. IEP review, document management, email templates, calendar sync, communication log.
Phase 3–4: Scale (Wk 21–48)
Covers journey stages 5–6: Deepening through Advocacy. Community, healthcare integration, insurance, legal, provider portal, analytics.
